About the Pangolin Simple Web Interface

Pangolin is an super simple open source web interface for OpanAI's Whisper automatic speech recognition (ASR) system. OpenAI Whisper. This web interface is designed to be simple and easy to use. All the source code is available on the Website and on GitHub.
